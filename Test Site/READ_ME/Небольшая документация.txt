Данный проект создавался в качестве тестового задания. 
Автор: komptip.
Все скрипты написаны собственноручно и специально для данного задания.
Время выполнения ~18 часов.
Папки:
Content - Используется для хранения контента (Изображений и т.д.)
Controller - Папка с контроллерами
Models - Папка с моделями
css - Папка с CSS
javascript - Папка для Javascript`ов. На момент окночания работы - пуста.
pages - Папка хранения всех страниц.
small_scripts - Небольшие скрипты, используемые для каких-либо не значительных функций.

Файлы в корне:
.htacess - htacess
index.php - Используется только для редиректа.
route.php - Файл, который отвечает за перемещение пользователя по сайту, 
в нем находятся почти все пути.

Вспомогательные скрипты (В папке Small_scripts):
db.php - Подключение к БД.
captcha.php - Генерирует капчу. 
taskconfimer.php - Добавляет задаче пометку  "Выполнено".
taskremover.php - Удаляет задачую
textchanger.php - Изменяет текст задачи.

База данных:
ID - Идентификатор, максимальная длинна: 11. INTEGER
email - Varchar, максимальная длинна: 255. По умолчанию: NULL
username - Varchar, максимальная длинна: 255. По умолчанию: NULL
text - Text, максимальная длинна: 10 000. По умолчанию: NULL
status - Varchar, максимальная длинна: 255. По умолчанию: "No"
changedbyadmin - Varchar, максимальная длинна: 255 По умолчанию: "No"
